# baby-tracker
A DIY baby tracking system that uses NFC tags, Google Sheets, Google Apps Script, Home Assistant, and AWTRIX to record feeding, diaper changes and sleep
## Features

- Record feeding, pee, poop, and sleep with NFC tags or shortcuts 
- Save all records to Google Sheets
- Show baby care data on a dashboard
- Display latest feeding/sleep info on AWTRIX
- Designed for family use
